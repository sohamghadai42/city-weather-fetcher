# Weather Checker 

A lightweight JavaScript function that fetches real-time weather data using the OpenWeatherMap API and alerts users about temperature conditions.

## Features

- Fetches current temperature for any city worldwide
- Displays temperature in Celsius
- Warns about extreme temperature conditions
- Simple error handling for invalid cities

## Prerequisites

- OpenWeatherMap API key (get one at [openweathermap.org](https://openweathermap.org/api))

## Setup

1. Clone this repository
2. Replace the `apikey` variable with your own OpenWeatherMap API key
3. Run the function with your desired city name

## API Reference

This project uses the [OpenWeatherMap API](https://openweathermap.org/current) to fetch weather data.
